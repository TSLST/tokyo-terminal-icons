### Long README Introduction

**Welcome to *tokyo-terminal***, a linux theme that transports your computer to a neon-colored terminal circa 1984.
This isn’t just a color scheme; it’s a high-contrast, high-energy workspace for **clarity and visual accentuation**.

Here are simple icon files that adapt to the general theme otherwise published for many open source softwares.
